| **思维模式**    | **背后故事**                                     | **核心启示**                 |
| ----------- | -------------------------------------------- | ------------------------ |
| **雨中的蚊子思维** | 蚊子在暴雨中不与雨滴硬碰，而是顺势调整姿态继续飞行。                   | 在逆境中学会借力、灵活应对，寻找突破机会。    |
| **消失的弹孔思维** | 二战时统计学家沃德发现，应强化飞机“没被打中的地方”，因为被击落的飞机才在这些部位中弹。 | 关注被忽略的信息，从未被看到的失败中学习。    |
| **还书思维**    | 孔子教导弟子：借书易，还书难；守信用者方为君子。                     | 借来的要还，重视责任与信用，是信任的根基。    |
| **灯塔思维**    | 水手在迷雾大海中靠灯塔指引前行。                             | 坚守长期目标与价值观，不被短期利益迷惑。     |
| **上游思维**    | 人们忙着救被河水冲下的孩子，有人提出应去上游查原因。                   | 不止解决结果，更要追溯并治理问题根源。      |
| **懒蚂蚁思维**   | “懒蚂蚁”不搬食物，而在观察路线，发现更高效的通路。                   | 真正的创新来自思考与优化，“聪明的懒”带来高效。 |
| **场域思维**    | 法拉第的“场”理论启发：个体行为受环境力量影响。                     | 关注系统与氛围的影响，改变“场”才能改变结果。  |

逆向思维

方位逆向：老公晚上不回家，提款机，小偷偷手机
结果倒推法：租店面，打玻璃，租房打听用途。丢借据，借2000，写借2500.草船借箭
心里逆反法：土豆到法国，用警卫守护。潘多拉魔盒，禁果效应。
转换型：2楼进电梯睡了别人老婆八年。提价，没有浪费奖励100元
缺点型：拍照321睁开眼睛。
反转型：让水离开人。买5斤水果，拿出2斤然后要这两斤。
